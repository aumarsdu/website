# 申请时间线生成器 产品需求文档 (PRD)

## 一、产品概述

### 1.1 背景与动机
*   **用户痛点**：留学申请流程复杂，不同国家/学段时间线差异大，家长和学生普遍焦虑"什么时候该做什么"。本工具通过输入基础信息即可获得清晰的个性化时间线，降低信息不对称。
*   **获客瓶颈**：过度依赖小红书内容+聚光投放，缺少"工具型"流量入口。本工具可承接SEO长尾词（"留学申请时间线"）自然流量+小红书笔记引流。
*   **转化断层**：用户从"看到内容"到"加微信"缺少中间触点。工具提供即时价值（80%），完整规划需加微（20%），自然过渡。
*   **数据缺失**：用户画像靠人工沟通获取，效率低。工具使用即采集（目标国家、学段、专业方向、当前年级），获取精准线索。

### 1.2 目标用户
1.  **高中生家长（美本/英本）**：预估占比40%（例："孩子高一，申美国Top30该怎么规划？"）
2.  **大学生本人（美研/英研）**：预估占比35%（例："大二了，想申CS硕士，现在来得及吗？"）
3.  **低龄规划家长（初中）**：预估占比15%（例："初二想走国际路线，什么时候开始准备？"）
4.  **留学顾问/同行**：预估占比10%（用作客户沟通辅助工具）

### 1.3 产品目标（MVP阶段）
*   月UV：≥ 500
*   工具完成率（输入→生成时间线）：≥ 60%
*   CTA点击率（加微/留资）：≥ 8%
*   工具→加微转化率：≥ 5%
*   生成时间线→分享率：≥ 3%

## 二、核心功能设计

### 2.1 输入参数设计
1.  **目标国家/地区（必填）**：美国 / 英国 / 中国香港（MVP阶段），其余标"即将上线"。
2.  **申请层级（必填）**：本科 / 硕士 / 博士。
3.  **当前年级（必填）**：初二 / 初三 / 高一 / 高二 / 高三 / 大一 / 大二 / 大三 / 大四 / 已毕业（根据申请层级动态展示）。
4.  **目标专业方向（必填）**：理工科 / 商科 / 社科人文 / 艺术设计 / 医学/生命科学 / 计算机/AI / 未确定。
5.  **目标院校层级（非必填）**：Top 10 / Top 30 / Top 50 / Top 100 / 不确定。
6.  **申请轮次（非必填）**：EA/ED / RD / 滚动录取 / 不确定（仅本科申请显示）。

### 2.2 输出：时间线内容模型
1.  **阶段名称**：时间区间 + 关键词（例："高一上学期（9-12月）· 探索与定位"）
2.  **核心任务**：该阶段必须完成的 3-5 项关键动作。
3.  **背景提升建议**：科研/竞赛/实习等建议（与业务强关联）。
4.  **关键截止日期**：标化考试、申请截止等硬性节点。
5.  **风险提醒**：该阶段常见的坑和误区。
6.  **服务锚点**：🔒 标记需加微获取的深度内容（例："🔒 获取适合你专业方向的科研项目推荐"）。

## 三、CTA 与转化设计

### 3.1 CTA 触发点
*   **时间线中间（第3-4阶段后）**：展示前 60% 内容后锁定，制造"看到一半想看完"的动力。文案："🔒 解锁完整规划 + 个性化背景提升方案"。
*   **每个阶段的背景提升建议**：直接衔接核心业务。文案："🔑 获取适合你的科研/竞赛项目推荐"。
*   **时间线底部**：文案："📋 下载完整 PDF 版时间线（含详细 checklist）"。
*   **分享图片水印**：文案："扫码生成你的专属留学时间线" + 二维码。
*   **"时间紧迫"提醒**：当用户年级较高、剩余时间不足时触发。文案："⏰ 你距离 XX 只剩 X 个月，立即获取加速方案"。
